Project A.R.O.W.W

Project A.R.O.W.W is an exciting real time multiplayer game built for my Capstone project at Neumont University. The game takes place through an Android App that lets users interact with others near them using the Google Maps API. Once near each other, players can chat, hack, or steal from one another. The goal of the game is to become hack people before they hack you while building an alliance with those around you.

Technologies Used

Xamarin | C# | Google Maps API for Android | Node.JS | Web Sockets
